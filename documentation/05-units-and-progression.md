# Units & Progression

## Unit Levels
- Each unit has a short level span (10–15 levels)
- Stats increase per level

## Quality / Tier System
Internal-only naming:
- Tier 1
- Tier 2
- Tier 3

Flavor titles are unit-specific and cosmetic.

## Gaining Levels
- Levels are gained by spending XP
- The only way a unit's level goes down is when it gets reset as part of promotion
- XP needed per level is determined by tier * target level * 50

## Gaining XP
- XP is tracked on a per unit basis
- XP is awarded as part of Combat and Boss encounters for any units that participate
- Awarded XP is determined based on sum of xp_reward from each enemy in the encounter

## Tier Advancement
- Combine 3 max-level units to advance tier
- Tier advancement:
  - Resets level to 1
  - Increases base stat floor
  - Tier 2 L1 ≈ Tier 1 L4–5 (slightly stronger)
  - Tier 3 L1 < Tier 1 max

## Special Requirements
- Tier 3 advancement requires:
  - Units + rare, region-specific item
