# Core Gameplay Loop

The core loop is intentionally short and repeatable.

1. Select Region
2. Explore Procedurally Generated Map
   - 5–8 encounters
   - ~75% combat encounters
   - Static "node" encounters mixed in
3. Boss Encounter (unlocked after map completion)
4. Loot Rewards
5. Advancement Phase
   - Spend XP
   - Equip dice
   - Combine units
6. Energy cost applied
7. Repeat

Energy is a real-world time-gated resource that limits daily progression.
