# Core Gameplay Loop

The core loop is intentionally short and repeatable.

1. Select Region
2. Energy cost applied
3. Explore Procedurally Generated Map
   - 5–8 encounters
   - ~75% combat encounters
   - Static "node" encounters mixed in
4. Boss Encounter (unlocked after map completion)
5. Loot Rewards
6. Advancement Phase
   - Spend XP
   - Equip dice
   - Combine units
7. Repeat

Energy is a real-world time-gated resource that limits daily progression.
