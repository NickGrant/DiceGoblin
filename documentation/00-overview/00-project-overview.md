# Dice Goblins - Project Overview

Status: active  
Last Updated: 2026-03-02  
Owner: Product  
Depends On: `documentation/00-overview/01-core-gameplay-loop.md`, `documentation/01-architecture/00-tech-stack.md`

Dice Goblins is a multiplayer browser-based tactical RPG focused on:
- Automated combat
- Squad-based progression
- Dice-driven unit customization
- Persistent, shared world systems with underhanded multiplayer interactions

The game is PvE-first. There is no direct PvP combat in MVP, but multiplayer interaction occurs through:
- Shared regions
- Investment/bribery systems
- Ally and betrayal mechanics
- PvP is not slated for MVP

The intended tone sits between:
- **Munchkin** (irreverent, comedic, goblin chaos)
- **Borderlands** (procedural loot, named modifiers, escalation)

This document describes the high-level vision and acts as an anchor for all technical and design decisions.
