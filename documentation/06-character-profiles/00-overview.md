# Character Profiles

## Purpose
- This directory stores creative character references used to guide art direction and flavor.
- These files are not implementation-authoritative gameplay contracts.
- Gameplay behavior, balance, and API/schema contracts remain defined in architecture and systems docs.

## Canonical Usage
- Treat profile files as art prompt inputs and visual tone references.
- Use `PROFILE_TEMPLATE.md` to keep new entries consistent and usable.
- Do not infer gameplay rules from profile prose unless separately documented in systems docs.

## Goblin Visual Design
- Design philosophy: "Industrial Scavengers"
- Visual identity: busy, clanking, asymmetrical salvage-tech silhouettes
- Materials: corrugated sheet metal, rusted pipes, leather straps, tire rubber, stained canvas
- Technology: mechanical; gears, pressurized steam, pistons, and ratcheting mechanisms
- Visual motif: hazard star and yellow-black caution striping
- Color palette:
  - Primary: dull green (skin)
  - Secondary: rust orange and oily black (metal)
  - Accent: hazard yellow (markings/paint)
- Physique: small, wiry, hunchbacked; large ears and pointed noses

## Kobold Visual Design
- Design philosophy: "Primal Mountaineers"
- Visual identity: lean, vertical, organic silhouettes with scavenger-tech from natural materials
- Materials: bone, cured hides, tortoise shell, feathers, obsidian glass, mountain-ash wood
- Technology: tension and gravity; sinew bows, staff-slings, and simple pulleys
- Visual motif: 80/20 hybridization of reptilian scales and canine frame
- Color palette:
  - Primary: dusty grey or earthy brown/red (skin/fur)
  - Secondary: bleached bone and obsidian teal
  - Accent: vibrant feather plumes (red/blue/yellow)
- Physique: lanky digitigrade climbers with wolf-like snouts and crocodilian scale/tail features
